MemoryGuard — IoT + AI Healthcare Dashboard for Alzheimer's Monitoring
A real-time web dashboard for caretakers to monitor Alzheimer's patients using Raspberry Pi 4, camera, GPS, GSM, and accelerometer modules.

Features

Live Dashboard — Real-time GPS map (Leaflet.js), fall detection status, last face recognition event, and device online/offline indicators
Person Recognition Panel — Upload photos of family/doctors/caretakers, view known persons grid, and browse recognition history with confidence scores
Alert Log — Full timeline of fall alerts, unknown person detections, and geofence breaches with GPS coordinates
Geofence View — Map with pre-drawn safe zone boundary; visual alert when patient exits the zone
Secure Login — Caretaker authentication via Supabase (email/password), protected routes


Hardware
Module                           Role
Raspberry Pi4              Central processing unit
Camera Module              Face recognition
GPS Module                 Real-time location tracking
GSM Module                 Sends alerts to caretaker
Accelerometer              Fall detection
